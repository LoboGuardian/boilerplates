# boilerplates

A curated collection of 72 project templates across 12 categories — frontend, backend, mobile, desktop (Windows, Linux, macOS), game dev, AI/ML, and infrastructure. Each boilerplate includes stack decisions, folder structure, and architecture notes to help you start any project with confidence.
